# Control_de_Temperatura
Sistema de control de temperatura por PWM de hardware utilizando el Timer 1 del ATmega2560 (Arduino Mega). Frecuencia variable por software, etapa de potencia con driver IR2110 y un MOSFET IRFZ44N para el manejo de carga resistiva a 12V, e interfaz visual en pantalla LCD 20x4 mediante bus de comunicación I2C.
